For each new project that you wish to work on initial git configuration is necessary. Please follow the procedure described below to correctly configure your project's git working environment.

Step 0.

Open MINGW terminal in the root directory of your project

Windows Git Gui (installed with git scm for windows) Open your project and then select "Repository" -> "Git Bash" from menu OR right click in your project's root folder and select "Open Git Bash here"

Windows Source Tree Open your project and then select "Actions" -> "Open in Terminal" from menu OR select "Terminal" from a toolbar on the right
